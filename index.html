Network map of competitive relationships between SP100 companies

Node sizes represent market caps.

Links represent competitive relationships between companies according to MarketWatch.com Identifying competitors is inherently subjective, from my research, MarketWatch seemed to have the most thorough competitor data.


A few quick observations:

The network has three main clusters: tech, pharmaceutical and financial.

MasterCard connects the finanical and tech clusters, MasterCard is arguably a tech company in that it has partnered with Google and AT&T on mobile payment products. Other credit card companies have not focus on mobile payment products to the same extent.











Technical details:
A Python script scrapes websites for SP100 companies, market cap sizes and competitive data. The Python library NetworkX generates a JSON file of nodes and links. The JavaScript D3 visualization library creates the graph. The D3 graph is adapted from http://bl.ocks.org/eyaler.
